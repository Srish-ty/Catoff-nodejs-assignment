# Steps to setup in local

1. Clone the repository

2.

```bash
cd project
```

3. Add variable in .env
   Follow sample.env format for the same.

4. Install dependencies:
   ```bash
   pnpm install
   ```
5. To run in local:
   ```bash
   pnpm run start
   ```
6. To run in watch mode:
   ```bash
   pnpm run start:dev
   ```
7. To build
   `pnpm run build`
